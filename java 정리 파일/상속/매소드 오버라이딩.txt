메소드 오버라이딩(Method Overriding)
- 상속관계의 클래스 간에 수퍼 클래스의 메소드를 서브 클래스에서 재정의 하는 것
	매소드의 이름. 매개변수 타입 및 개수, 리턴 타입등 모든 것을 동일하게 작성
	즉/ 상위 클래스의 함수이름과 하위 클래스의 함수이름이 같을 때

- 수퍼 클래스의 메소드 대신 서브 클래스에 오버라이딩된 메소드가 무조건 실행
	동적 바인딩(dynamic binding)
		실행시간에(runtime) 재정의 되는 것(프로그램이 실행되면서 이루어짐)
		함수 이름과 주소사이를 느슨하게(변수 처럼 결정되지 X)하는 것 -> 주소 변경 가능
