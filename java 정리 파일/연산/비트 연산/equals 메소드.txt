	* equals 와 ==
		공통점 : boolean type으로 반환
		차이점 :
		1) 형태
			equals() : 메소드
			== : 연산자
		2) 비교 대상
			equals() : 대상의 내용 자체
			== : 주소값

		ex) 똑같은aaa이나 equals는 true, ==은 false를 출력
		String a = "aaa";		-> 주소값 : 100이라 하자
		String c = new String ("aaa");	-> 주소값 : 300아러 하자
